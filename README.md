
# Excel Sales Analysis Dashboard Project

## Project Description
This project analyzes sales data using Microsoft Excel to generate insights and interactive dashboards for business decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP / XLOOKUP
- Charts and Graphs

## Steps Performed
- Data cleaning (removed duplicates, handled missing values)
- Created Pivot Tables for sales analysis
- Built interactive dashboard with charts and slicers
- Analyzed sales by region, product, and month

## Key Insights
- Top-performing products and regions identified
- Monthly sales trend analysis
- High revenue categories highlighted
